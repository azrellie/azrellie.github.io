Thermodynamic equations:

Relative Humidity:

Where:

RH = Output

TD = Dew point in Celcius

T = Air temperature in Celcius

exp = Exponent function

RH = 100 * (exp((17.625 * TD) / (243.04 + TD)) / exp((17.625 * T) / (243.04 + T)))

Dew Point:

Where:

TD = Dew point in Celcius

T = Air temperature in Celcius

RH = Relative Humidity

log = Logarithmic function

TD = 243.04 * (log(RH / 100) + ((17.625 * T) / (243.04 + T))) / (17.625 - log(RH / 100) - ((17.625 * TD) / (243.04 + TD)))

Air Temperature:

Where:

TD = Dew point in Celcius

T = Air temperature in Celcius

RH = Relative Humidity

log = Logarithmic function

T = 243.04 * (((17.625 * TD) / (243.04 + TD)) - log(RH / 100)) / (17.625 + log(RH / 100) - ((17.625 * TD) / (243.04 + TD)))

Heat Index:

Where:

T = Air temperature in Fahrenheit

RH = Relative Humidity

HI = Heat index in Fahrenheit

HI = -42.379 + 2.04901523*T + 10.14333127 * RH - 0.22475541 * T * RH - 0.00683783 * T * T - 0.05481717 * RH * RH + 0.00122874 * T * T * RH + 0.00085282 * T * RH * RH - 0.00000199 * T * T * RH * RH

If RH <13% and T > 80 and T < 112 then subtract from HI with this amount

ADJUSTMENT = (13 - RH) / 4 * sqrt((17 - ABS(T - 95)) / 17)

If RH >85% and T > 80 and T < 87 then add from HI with this amount

ADJUSTMENT = ((RH - 85) / 10) * ((87 - T) / 5)

Specific Humidity:

Where:

TD = Dew point in Celcius

P = Air pressure in hPA or mBar

a = Variablized equation for readability

SH = Specific Humidity in g/kg

a = 6.112 * exp((17.67 * TD) / (TD + 243.5))

SH = (0.622 * a) / (P - (0.378 * a)) * 1000

Absolute Humidity:

Where:

TD = Dew point in Celcius

RH = Relative humidity

AH = Absolute Humidity in g/m3

pow = Exponent power function

AH = 6.112 * pow(2.71828, ((17.67 * TD) / (TD + 243.5))) * RH * 2.1674 / (273.15 + TD)

Cloud base:

Where:

T = Air temperature in Fahrenheit

TD = Dew point in Fahrenheit

CB = Cloud base in feet

CB = (T - TD) / 4.4 * 1000

Lapse Rate:

Where:

TD = Dew point in Celcius

EL = Air temperature in Celcius at 12000 feet

a, b, c, d = Variablized equation for readability

exp = Exponent function

LP = Lapse rate in C/km

EL = AirTempAtElevation(T, 12000)

a = 2510 - 2.38 * EL

b = (6.11 / 1012.34) * exp(0.622 * a / 0.287 / 273 / (1 * TD + 273) * TD)

c = 0.622 * a * b / (0.287 * (1 * TD + 273) * (1 * TD + 273))

d = 10 * (1 + 0.622 * a * b / (1 * 0.287 * (1 * TD + 273))) / (1 + (0.622 * a * c) / (1.005 * 1))

LP = d

CAPE:

Where:

P = Air pressure in hPa or mBar

TD = Dew point in Celcius

VT = Virtual temperature in Kelvin

PT = Air temperature in Kelvin at 12000 feet

VTC = Virtual temperature in Celcius

pow = Exponent power function

VT = (TD + 273.15) / (1 - 0.379 * (6.11 * pow(10, ((7.5 * TD) / (237.7 + TD))) / P))

PT = AirTempAtElevation(VT, 12000)

VTC = KtoC(VT)

CAPE = 9.81 * ((VT - VTC) / VTC)

Where:

W = Surface winds in m/s

SP = Speed shear in m/s

SP = W / 3
